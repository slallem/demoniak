# Corpus source

These 12 files are the individual stories of *The Adventures of Sherlock Holmes* by Arthur Conan Doyle, split from **Project Gutenberg eBook #1661** (https://www.gutenberg.org/ebooks/1661).

The work is in the **public domain**. The Project Gutenberg license header/footer has been stripped so only the literary text remains; each file keeps a one-line provenance note.
